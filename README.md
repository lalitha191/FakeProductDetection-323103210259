## 📌 Project Description
This project detects fake products using blockchain technology.

Each product is stored with a unique ID on Ethereum.  
A QR code is generated for verification.  
Users can check if a product is genuine or fake.

---

## 🚀 Features
- Add product to blockchain
- QR code generation
- Verify product authenticity
- Detect fake products
- MetaMask + Ganache integration

---

## 🛠️ Technologies Used
- Solidity
- Truffle
- Ganache
- Web3.js
- HTML, CSS, JavaScript

---

## ▶️ How to Run
1. Start Ganache  
2. Run: `truffle migrate --reset`  
3. Open `index.html`  
4. Connect MetaMask  
5. Add & verify products
